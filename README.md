Food Delivery App SQL Project :

A complete, beginner-friendly yet professional SQL project based on a Food Delivery App (Swiggy/Zomato style).
This project demonstrates real-world database design, SQL commands, joins, subqueries, views, DCL, TCL, and more.

*Project Overview

This project simulates a real food delivery ecosystem where:
Customers place food orders
Restaurants offer menu items
Delivery partners deliver food
Customers rate restaurants
The goal is to help learners practice SQL in a real-world scenario with a clean, simple, and easy-to-understand database.
